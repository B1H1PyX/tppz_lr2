Інформаційна система оцінювання поточного стану динамічного об’єкту Information System for Assessing the Current State of a Dynamic Object

Розробка алгоритму та програмної системи для непрямої діагностики динамічних об’єктів за інтегральними інформаційними характеристиками спектру контрольованих сигналів.

Актуальність теми

Працюючі механізми — двигуни, редуктори, коробки передач — породжують віброакустичні сигнали, аналіз яких дозволяє оцінювати поточний технічний стан об’єкта без використання спеціалізованих датчиків. Своєчасна діагностика дозволяє запобігати аваріям та зменшувати витрати на обслуговування.

Постановка задачі

В умовах відсутності спеціалізованих датчиків виникає потреба кількісно оцінювати відхилення поточного стану динамічних об’єктів від нормального за контрольованими сигналами, що формуються під час їх роботи або при імпульсному впливі.

Мета та завдання дослідження

Мета

Розробити алгоритм та комп’ютерну програму для оперативного оцінювання поточного стану динамічних об’єктів за спектральними характеристиками контрольованих сигналів.

Основні завдання

  • Проаналізувати методи спектральної діагностики
  • Реалізувати розкладання сигналу в ряд Фур’є
  • Обчислити інтегральні інформаційні оцінки спектру
  • Дослідити чутливість оцінок до зміни параметрів об’єкта
  • Перевірити алгоритм на моделі об’єкта 2-го порядку

Методологія дослідження

Аналіз сигналів здійснюється у частотній області шляхом застосування швидкого перетворення Фур’є (БПФ). Для оцінювання поточного стану динамічного об’єкта використовуються інтегральні інформаційні характеристики спектру, що є чутливими до зміни параметрів системи.

Очікувані результати

В результаті роботи буде отримано програмну систему, що дозволяє виявляти відхилення поточного стану динамічних об’єктів від нормального за зміною інтегральних спектральних оцінок.

Об’єкт дослідження

Роботу алгоритму продемонстровано на прикладі динамічного об’єкта другого порядку, зокрема вібротранспортної системи.
